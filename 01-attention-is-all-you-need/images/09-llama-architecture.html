Each Change Explained

1. Decoder-Only (drop encoder + cross-attention)

The original Transformer was built for translation: an encoder reads French, a decoder writes English. Modern LLMs don't have a separate "input" and "output" — everything is a sequence of tokens. The prompt and the completion are one continuous stream. The causal mask in the decoder is the only structure needed. Early layers learn to represent/understand (encoder's job), later layers learn to predict (decoder's job). This also eliminates the cross-attention layer (~1M params/layer), simplifying the architecture.

2. Pre-RMSNorm (instead of Post-LayerNorm)

Original (Post-LN): output = LayerNorm(x + Sublayer(x))
LLaMA (Pre-RMSNorm): output = x + Sublayer(RMSNorm(x))

Pre-norm normalizes the input BEFORE the sublayer, then adds the residual. This keeps the residual stream "clean" — information flows directly through skip connections without being distorted by normalization. Training is much more stable at large scale (no need for careful warmup).

RMSNorm vs LayerNorm: LayerNorm does mean subtraction + variance normalization. RMSNorm only does root-mean-square normalization (no mean subtraction, no bias). Simpler, 10-15% faster, same performance.

3. RoPE — Rotary Position Embeddings

Original: add PE to embeddings once, before layer 1 x = Embedding(token) + PE(position)
LLaMA: rotate Q,K vectors at every attention layer q'_m = R(m) · q_m    k'_n = R(n) · k_n
q'_m · k'_n = q_m · R(m-n) · k_n ← depends only on relative position (m-n)!

RoPE rotates pairs of dimensions by angles that depend on the position. The dot product between rotated Q and K naturally encodes the relative distance between tokens, not their absolute position. This means a model trained on 4K context can be extended to 128K+ with minimal finetuning (RoPE frequency scaling). The original sinusoidal encoding was absolute and didn't extrapolate.

4. Grouped-Query Attention (GQA)

MHA (original): every head has its own Q, K, V 32 Q heads × 128d, 32 K heads × 128d, 32 V heads × 128d
GQA (LLaMA 2 70B, LLaMA 3): Q heads share K,V across groups 32 Q heads × 128d, 8 K heads × 128d, 8 V heads × 128d
4 Q heads share 1 K head and 1 V head → 4× less KV-cache

During autoregressive inference, the model stores all previous tokens' K and V tensors (the "KV-cache"). For a 70B model with 80 layers, this cache is gigabytes. GQA reduces K/V heads from 64 to 8 → cache shrinks 8×, inference throughput increases dramatically, with negligible quality loss.

5. SwiGLU (instead of ReLU FFN)

Original FFN: FFN(x) = max(0, xW₁ + b₁) · W₂ + b₂   (2 matrices, ReLU)
LLaMA SwiGLU: SwiGLU(x) = (xW₁ ⊙ Swish(xWgate)) · W₂   (3 matrices, no bias)
Swish(x) = x · σ(x) — smooth gating, no dead neurons

ReLU kills any negative input to exactly 0 — "dead neurons" that never activate and never learn. SwiGLU uses a smooth gate (Swish) that allows small negative values through, plus a multiplicative gating mechanism that lets the network learn which neurons to activate. The third weight matrix (Wgate) adds ~50% more FFN params, so d_ff is reduced from 4× to ~2.67× d_model to keep total params similar. All bias terms are removed.

6. No Bias Terms Anywhere

LLaMA removes all bias vectors from linear layers, attention projections, and normalization. The rationale: at scale, biases add negligible expressivity but add complexity to quantization and parallelism. Removing them simplifies the architecture with no measurable quality loss.

One LLaMA Decoder Block (Pseudocode)

# Input: x of shape (B, T, 4096)

# Sub-layer 1: Attention
h = RMSNorm(x)                  # pre-norm → (B, T, 4096)
Q = h · WQ                      # (B, T, 4096) → (B, T, 4096) → reshape (B, 32, T, 128)
K = h · WK                      # (B, T, 4096) → (B, T, 1024) → reshape (B, 8, T, 128)
V = h · WV                      # (B, T, 4096) → (B, T, 1024) → reshape (B, 8, T, 128)
Q, K = apply_rope(Q, K, pos)     # rotate Q,K by position-dependent angles
K, V = repeat_kv(K, V, n_rep=4) # expand 8→32 heads by repeating for GQA
attn = softmax(Q · KT / √128) · V # (B, 32, T, T) × (B, 32, T, 128) → (B, 32, T, 128)
attn = reshape(attn) · WO      # (B, T, 4096) → (B, T, 4096)
x = x + attn                    # residual connection

# Sub-layer 2: Feed-Forward (SwiGLU)
h = RMSNorm(x)                  # pre-norm → (B, T, 4096)
gate = swish(h · Wgate)        # (B, T, 4096) → (B, T, 11008)
up = h · Wup                    # (B, T, 4096) → (B, T, 11008)
ffn = (gate ⊙ up) · Wdown        # (B, T, 11008) → (B, T, 4096)
x = x + ffn                     # residual connection

# Output: x of shape (B, T, 4096) — same shape in, same shape out
