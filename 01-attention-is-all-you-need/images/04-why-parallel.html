1 parallel operation. QKT is a matrix multiply — GPU computes all 16 output cells simultaneously. With S = 100,000 tokens, it's still one matmul (though memory is S² = 10 billion entries per head).

The core insight: The parallelism isn't magic — it's that the Transformer reformulates everything as matrix multiplications. GPUs are literally designed to compute all output cells of a matrix multiply simultaneously. An RNN forces serial dependency (h_t needs h_{t-1}), which means most GPU cores sit idle. The Transformer has no such dependency — every token's attention computation is independent and can happen at once.

The tradeoff: Memory scales as O(S²) for attention. At S = 200,000, that's 40 billion entries per head × 8 heads = 320B entries. This is why FlashAttention (Dao et al., 2022) was needed to make long-context models practical.
